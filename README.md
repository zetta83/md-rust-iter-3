# Проект модуля 3

В этом проекте вы создадите полноценную систему блога, которая объединяет все изученные технологии: веб-сервер с HTTP и gRPC API, клиентскую библиотеку, CLI-инструмент и WASM-фронтенд. Система включает регистрацию и аутентификацию пользователей через JWT-токены, что позволяет защитить операции создания, редактирования и удаления постов; просмотр списка и отдельных постов — без входа.

**Плановое время на выполнение:** 15 часов, включая время на оформление и тестирование.
